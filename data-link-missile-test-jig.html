Data Link Missile Test Jig

Data Link Missile Unit Test Jig is a combination of three modules; Transmitter, Dual Channel Receiver and DTM. Input of 300 MHz from DTM is up converted to1400 MHz signal by using the fixed frequency synthesizer and further converted to transmitting frequency by using Variable frequency synthesizer of 6 to 6.4 GHz. Receiver has two identical channels, which takes 4.4 to 4.5 GHz input and is down converted into 1.3GHz with the variable frequency synthesizer and further to 70MHz with fixed frequency synthesizer.

Data Link Missile Test Jig Features and Applications

Features

  • DTM output of 300 MHz.
  • Dual stage up conversion.
  • Dual stage down conversion.
  • Variable frequency synthesizers.
  • Fixed frequency synthesizers.
  • Message signal consists of missile parameters.
  • EMI/EMC compliant.
  • Meets the MIL-461E/F.

Applications

  • Test system for simulating missile parameters.
