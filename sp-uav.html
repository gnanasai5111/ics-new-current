SP UAV

SP UAV for Secondary Surveillance Radar (SSR) is indigenous development program for Indian Army. The SSR System is capable of surveillance and tracking while on the halt and move on short halts for providing Air Defense protection to mechanized formations. The SP UAV is designed on Single Board Computer architecture with high end processors and onboard ADC and high-end FPGAs. The signal processor of Transponder shall be configured with feature of DPSK and MSK demodulation in addition to the conventional pulse amplitude demodulation.

SP UAV Features and Applications

Features

  • Zynq Ultra scale FPGA: Arm Cortex XCZU11EG-2FFVC1156I (FPGA + SoC).
  • RF Sampling ADC of 3GSPS is Used for direct down conversion of 1.2GHz Signal.
  • 4GB DDR4 Memory to PS and 2GB DDR4 memory to PL are used.
  • 256MB of QSPI Flash used for data storing/Application.
  • 16GB eMMC NAND is used for memory and Data storage.
  • USB3.0/USB2.0 Interface and SD Card Slot for data storage.
  • 10/100/1000M Bit Ethernet Transceiver is used.
  • Ethernet interface routed to VPX connectors.
  • MIL1553B Interface is used for the Communication Interface.
  • ARINC 429 Interface is used for the data Transfer.
  • 400Hz Audio Generator is used with 100khz filter.
  • 24V DC power Input to the board and required voltages are generated on Board.
  • The board is operated in industrial grade temperature.
  • JTAG Interface for on board debugging with 14 pin connector.
  • PCI Express interface routed to VPX connectors.
  • The Total Board is enclosed in a 3U VPX chassis.

Applications

  • RADAR Surveillance
