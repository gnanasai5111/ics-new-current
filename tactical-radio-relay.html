ICS Tactical Radio Relay Link (TTR-B3) is a Line-of-Sight (LOS) radio link operating in the Band III (1350 MHz to 2690 MHz) Frequency Band and is intended for use in Tactical Network for long haul transmission of data traffic. The Radio Relay Link is configured in two parts. 1. Outdoor Unit (ODU) on a top of a telescopic mast along with Antennas and Antenna Alignment Unit (AAU). 2. An Indoor Unit (IDU) for two WAN connections, control and monitoring of Radio Link. The distance between the IDU and ODU can be upto 2 Km using an optical fiber link. It is capable of Full-Duplex Data transfer up to 34 Mbps. It can be deployed for Point to Point (PTP) Communication link using Narrow Beam Antennas, at Data rates ranging from 0.25 to 34 Mbps. Frequency Hopping is supported for data rates upto 2 Mbps. This Radio Link supports two independent WAN interfaces, which are configurable to serve legacy TDM traffic and Ethernet Data traffic respectively.

Configuration & Features

  • Indoor Unit (IDU).
  • Outdoor Unit (ODU).
  • Telescopic Mast.
  • Antenna Alignment Unit.
  • Parabolic grid reflector Antenna with high-side and back-lobe attenuation.
  • Power Supply Units (PSU).
  • Cables and Accessories.
  • GPS for location identification.
  • User data rates and waveforms are Software configurable and upgradable.
  • Total 99 programmable positions for any FF pairs or FH bands.
  • Interface: Ethernet 10/100 Base T.
  • E1, E3 and Ethernet base band interfaces, with both Electrical and Optical connectivity.
  • Transmission Capacity: 0.25 to 34 Mbps.
  • ECCM: Adaptive and Conventional FH Modes.
  • Modem Modulations: BFSK, 16 QAM, 64 QAM.
  • Enhanced Burst Noise Robustness (QAM Noise Robustness).
  • AES Encryption.
  • Net Data Carrier System for ease of radio parameter setting.
  • Power On Self Test (POST).
  • Built In Test Equipment (BITE).
